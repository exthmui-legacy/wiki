欢迎来到 exTHmUI Wiki!

你可以在我们的 Wiki 中找到各种信息、如何编译和更多内容。

对于用户

下载 exTHmUI！

  • 你可以在这里下载到我们 官方支持设备 的 exTHmUI。

反馈 Bug！

  • 对于反馈 Bug 所需要的信息, 你可以 安装 ADB 然后 抓取 Log。
  • 然后通过阅读我们的 如何反馈 Bug 来学习如何反馈 Bug。

对于任何人

贡献 Wiki！

我们的 Wiki 基于 Jekyll.

  • 你可以在 GitHub 上查看源代码。
  • 你可以通过 引导界面 所说明的方法来完善维基。

为你的设备编译！

  • 每个官方支持的设备都有一套全面的编译说明。只需从我们的 支持设备 中选择你的设备就可以开始了。

对于开发者

申请成为官方维护者

  • 如果你是一位非官方维护者, 并且足够稳定，你可以按照我们的说明 提交一份申请。你提交申请后我们的开发人员会与你联系。
